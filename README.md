# CV of Benjamin Wilhelm

The curriculum vitae of Benjamin Wilhelm as webpage and PDF.

## Getting Started

### Dependencies

Make sure you have ruby and bundler installed.

Install the dependencies:

```
$ bundle install --path vendor/bundle
```

### Building

```
$ bundle exec rake
```
